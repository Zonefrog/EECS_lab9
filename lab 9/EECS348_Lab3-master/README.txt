To run each of these programs, simply follow the prompts on screen.

For program 1, add each of the 12 month's values to the text file named inputs, then run the program. Extra values will be ignored.

Fir program 2, each combination will be displayed for each way to score the given score. Any value less than 2 will cause the program to exit.

It is not advised that large values are put into the program, they will cause the program to process for a long time with the vast number of combinations that are possible for large scores.
